# このソフトウェアについて
このソフトウェアは、空想世界などの戦記録における戦力配置などを配置・再現することでその状況を同時に共有できる支援システムを提供します

# 使い方(開発環境)
ターミナルが2つ必要です。
1つ目のターミナル
`npm run dev`

2つ目のターミナル
`node ./index.js` (または `cd server && node index.js`)

# 本番環境構築

本番環境(Production)として動作させるための手順です。

## 1. フロントエンドのビルド
プロジェクトのルートディレクトリで以下のコマンドを実行し、静的ファイルを生成します。
```bash
npm run build
```
これにより `dist` フォルダに最適化されたファイルが生成されます。

## 2. バックエンドのセットアップと起動
バックエンド(Socket.IOサーバー)は `server` ディレクトリ内にあります。

1. サーバーディレクトリへ移動: `cd server`
2. 依存関係のインストール: `npm install`
3. サーバーの起動: `node index.js`

サーバーはポート `3001` で起動します。

## 3. フロントエンドの配信
`dist` フォルダの内容をWebサーバー(Nginx, Apacheなど)で配信するか、Viteのプレビュー機能を使用して確認します。
```bash
npm run preview
```

※ 本番運用時は、フロントエンドからバックエンド(ポート3001)へのWebSocket接続が可能なようにネットワーク設定を行ってください。



# ソフトウェアの使い方(アクセスなど)
